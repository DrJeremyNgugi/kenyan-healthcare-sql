# kenyan-healthcare-sql

SQL practice and reference queries for hospital data analytics in the Kenyan healthcare context.

## Purpose

This repository documents my SQL learning journey, with exercises and reference patterns tied to a realistic Kenyan hospital data model. Queries assume the patient/encounter/claim schema used in Mirai Analytics — a healthcare data activation platform I'm building for mid-sized Kenyan private hospitals.

## Structure

- `sql/practice_queries.sql` — Annotated queries from working through Jose Portilla's SQL Bootcamp, applied to the Mirai data model
- `sql/reference/` — Reusable query patterns for common hospital analytics questions (coming soon)

## About

I'm a practicing clinician in Nairobi and the founder of [Mirai Health Group](https://github.com/mirai-health-group). This repo is part of a five-month structured technical build I'm doing in parallel with running my clinical practice.
